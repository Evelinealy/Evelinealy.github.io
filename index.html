Hi there! My name is Aly. I am a full-stack developer in California.


Ever since I was a kid I enjoyed programming, especially when I used to play games like Minecraft and found out how to modify it to my pleasure.


I started taking programming seriously in my freshman year of High School when I signed up for my first Introduction to Computer Science, which happened to be a college course. It was the first time I really learned about the concept and truly loved it. After that, I went to a summer camp for coding hosted at Stanford College and created my very first game written fully in Java.


Thanks to the fun experiences in both my childhood and summer camp experiences, I decided to pursue my Bachelor's Degree in Computer Science. From my time studying in college, I decided that being a full-stack developer would be my goal in starting my career. My long-term goal is to go into Cyber Security and use my skills of making web appliacations to web application security. Since then, I have been polishing my skills in Javascript and Node.JS outside of school times and have made some very fun projects. I have discovered too, that I love designing and creating websites, so I have been refining my HTML and CSS knowledge as well.

Description

This project scenario was designed from an idea from my school.
